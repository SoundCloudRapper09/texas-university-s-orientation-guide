# Texas College Orientation Guide

A school-specific, timeline-driven orientation checklist for every major Texas university — built from each school's official admitted-student and orientation pages.

**Live site:** https://YOUR-USERNAME.github.io/texas-orientation-guide

## Schools covered
- UT Dallas (UTD)
- UT Austin
- Texas A&M
- Texas Tech
- UNT Denton
- TCU
- Baylor
- University of Houston

## Features
- **Real 2026 orientation dates** sourced from each school's official site
- **Timeline view** — tasks ordered by urgency across 4 phases, with date windows that calculate backwards from your chosen orientation date
- **Checklist view** — organized by category (accounts, financial aid, health, housing, academics)
- **Class research tasks** specific to each school's catalog and scheduling systems
- **Progress meter** tracking completion across all tasks

## How to deploy to GitHub Pages

1. Create a new GitHub repository named `texas-orientation-guide`
2. Upload `index.html` to the repository
3. Go to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Select `main` branch and `/ (root)` folder
6. Click **Save**
7. Your site will be live at `https://YOUR-USERNAME.github.io/texas-orientation-guide` within ~60 seconds

## To update
Edit `index.html` directly in GitHub's web editor (click the file → pencil icon) or push changes from your local machine.

## Sources
All checklist items sourced from official university orientation and admitted-student pages. Dates and requirements change — always verify at your school's official site.
